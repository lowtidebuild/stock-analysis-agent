# v1.1.0 Release Notes

## English

v1.1.0 makes the stock analysis agent more reliable, safer, and easier to trust in everyday use.

### What Improved

- **More dependable answers**  
  The agent is better at checking whether numbers, dates, sources, and conclusions actually fit together before presenting an investment view. This reduces polished-looking reports that contain hidden inconsistencies.

- **Less guessing when data is weak**  
  When a number cannot be verified well enough, the agent now prefers leaving it blank or flagging the uncertainty instead of filling the gap with a confident-looking estimate.

- **Clearer source confidence**  
  Reports now make it easier to understand whether the analysis is based on strong primary data, structured market data, fallback sources, or limited web evidence.

- **Safer use of web and fetched content**  
  Content collected from the web is treated more carefully, so untrusted text is less likely to steer the agent’s reasoning or sneak instructions into the workflow.

- **Better handling of Korean and U.S. data paths**  
  The workflow is more consistent across U.S. and Korean stocks, including improved handling of fallback data sources and Korean disclosure data.

- **More reliable saved analysis history**  
  Saved analysis snapshots are easier to reuse later without mixing old run files with current results.

- **Cleaner review flow before delivery**  
  The agent now separates mechanical checks from judgment-based critique, so reports can catch basic issues faster while keeping human-style investment reasoning focused where it matters.

- **Faster and leaner analysis context**  
  The agent passes a more compact evidence package into the main analysis step, helping reduce unnecessary context while keeping the facts needed for the final view.

- **Improved quick briefings**  
  Lightweight Mode A outputs still keep essential safeguards such as source confidence, date visibility, disclaimer checks, and blank-over-wrong behavior.

### Why It Matters

This release is focused on trust. The agent should now be less likely to overstate weak data, less likely to produce internally inconsistent reports, and better at explaining uncertainty in a way users can act on.

## 한국어

v1.1.0은 주식 분석 에이전트를 더 믿고 쓰기 쉽도록 안정성, 안전성, 결과 품질을 개선한 릴리즈입니다.

### 개선된 점

- **답변 신뢰도 향상**  
  투자 의견을 내기 전에 숫자, 날짜, 출처, 결론이 서로 맞는지 더 꼼꼼히 확인합니다. 그럴듯해 보이지만 내부적으로 앞뒤가 맞지 않는 보고서를 줄이는 데 초점을 맞췄습니다.

- **불확실한 데이터에 대한 추측 감소**  
  충분히 검증되지 않은 숫자는 억지로 채우지 않고, 비워 두거나 불확실성을 표시하도록 개선했습니다.

- **출처 신뢰도 이해가 쉬워짐**  
  분석이 공시 같은 강한 1차 자료에 기반한 것인지, 구조화된 시장 데이터인지, 보조 출처나 제한적인 웹 근거인지 더 명확히 구분할 수 있습니다.

- **웹에서 가져온 내용 처리 안전성 강화**  
  외부에서 수집한 텍스트가 에이전트의 판단이나 작업 지시를 오염시키지 않도록 더 조심스럽게 다룹니다.

- **미국/한국 종목 워크플로 일관성 개선**  
  미국 종목과 한국 종목 모두에서 fallback 데이터, 한국 공시 데이터, 실행 경로 처리가 더 안정적으로 맞춰졌습니다.

- **저장된 분석 기록 재사용 안정성 향상**  
  과거 분석 스냅샷을 다시 불러올 때 이전 실행 파일과 현재 결과가 섞일 가능성을 줄였습니다.

- **보고서 전달 전 검토 흐름 개선**  
  단순한 형식/계산 검사는 자동으로 더 빠르게 잡고, 투자 판단이 필요한 비평은 별도로 다루도록 정리했습니다.

- **분석 컨텍스트 경량화**  
  최종 분석 단계에 필요한 핵심 근거만 더 작고 정리된 형태로 전달해, 불필요한 정보로 판단이 흐려지는 문제를 줄였습니다.

- **빠른 브리핑 품질 보강**  
  짧은 Mode A 브리핑에서도 출처 신뢰도, 기준일, 고지문, 검증 불가 데이터 비움 처리 같은 기본 안전장치를 유지합니다.

### 왜 중요한가

이번 릴리즈의 핵심은 신뢰입니다. 에이전트가 약한 데이터를 과장하지 않고, 내부적으로 모순된 보고서를 줄이며, 사용자가 실제로 판단할 수 있는 방식으로 불확실성을 드러내도록 개선했습니다.
